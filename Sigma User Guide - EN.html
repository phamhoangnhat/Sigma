User Guide for Sigma Vietnamese Input Method 2.1.0

1. Introduction

Sigma is a modern Vietnamese input method editor (IME) for Windows 10 and Windows 11 (64-bit), developed by Phạm Hoàng Nhật. It is designed to overcome common limitations of other IMEs, providing a more flexible, intelligent, and optimized typing experience.

Key Features of Sigma:

Quickly switch between English | Vietnamese modes with a hotkey. The current word will automatically adapt to the new mode without needing to delete and retype.

Temporarily switch between English and Vietnamese within the current word, useful for writing Vietnamese text mixed with English terms.

Supports multiple input styles such as Telex, VNI, and Tích hợp (Integrated), suitable for shared computers with multiple users.

Provides many hotkeys to quickly change modes and configurations, helping to boost productivity.

Allows per-application configuration (Thiết lập cấu hình), reducing the need for manual switching when working across different software.

Integrates AI for smarter and more flexible language processing.

Important Notes for New Users:

In Sigma, settings such as English | Vietnamese mode or character set are stored per application and are automatically applied when switching between windows. Unlike most IMEs, you will not see an E or V indicator on the Sigma tray icon to switch modes, and you cannot open a global configuration panel from the tray icon to apply settings across all apps. Therefore, during initial use, you need to configure these options for each application you work with.

Sigma’s hotkey mechanism is unique: you press and release the Shift key first, then press and release the main key. This is not a simultaneous key combination like conventional shortcuts, allowing greater flexibility in customizing hotkeys while minimizing conflicts with other software.

2. Installing and Updating Sigma

  • Double-click SigmaSetup.exe to install. (No admin privileges are required.)
  • Download SigmaSetup.exe

On startup (if an Internet connection is available), Sigma will automatically check for updates and install the latest version when available.

3. Input Modes and Language Switching

Input Modes

  • English: Plain typing, no diacritic processing.
  • Vietnamese: Full Vietnamese typing using Telex, VNI, or Integrated mode.

Switching Modes

  • Left Ctrl + Left Shift: Toggle between Vietnamese and English.
  • Left Ctrl: Temporarily switch to the other mode for the current word being typed.

When switching modes, the word currently being typed will automatically adapt to the new language — no need to delete and retype.

Input modes are saved and automatically restored when switching between application windows, reducing the need for repeated mode changes when working with multiple apps.

Status Icons

  • Pink–orange icon with E: English
  • Green icon with V: Vietnamese

Notes

  • Left Ctrl and Left Shift refer to the keys on the lower-left side of the keyboard.
  • Keys must be pressed and released quickly to take effect:
    • Ctrl + Shift: within 1 second
    • Ctrl alone: within 0.3 seconds
  • If the key is held too long, the action will be ignored.

4. Character Sets & Vietnamese Input Styles

Character Sets

  • Unicode: Default and recommended.
  • VNI Windows: For documents using fonts with the VNI- prefix.
  • TCVN3 (ABC): For documents using fonts with the .VN prefix.

Input Styles

  • Telex, VNI: The two most common input styles.
  • Tích hợp: A style in Sigma that combines Telex and VNI, and also allows user customization.

Using Shortcuts

To quickly change settings, you use the Shift key combined with a specific shortcut key:

  • Press and release the Shift key quickly (within 0.3 seconds).
  • Immediately after, press the corresponding shortcut key to change the setting.
  • When the setting is successfully changed, a notification icon will appear in the middle of the screen as confirmation.

Example: To change the current character set, press and release Shift, then press M repeatedly to cycle through the sets: Unicode, VNI Windows, and TCVN3 (ABC).

These shortcut keys are the default of Sigma. Users can change them or disable shortcuts if desired by clicking the Sigma icon in the system tray and selecting Trình quản lý phím tắt.

Explanation of Configuration Options

Bộ mã

Allows switching between Vietnamese character sets such as Unicode, VNI Windows, and TCVN3 (ABC) to ensure compatibility with the software or system you are using.

In software that requires working with multiple character sets, you can use the shortcut: press Shift, then press M repeatedly to cycle through the sets. Each press of M switches to the next set.

When the character set is changed, the word currently being typed will also automatically switch to the new set immediately — saving time without needing to delete and retype.

Tác vụ AI

Allows you to choose a default AI task for each application.

To execute a task, simply select the typed text, then press Shift, followed by `. Sigma will automatically perform the corresponding AI task on the selected content.

Sử dụng chế độ tiếng Việt chủ động

When this mode is enabled, Sigma defaults to English input. To switch to Vietnamese, you need to press Left Ctrl + Left Shift.

During Vietnamese typing, if you perform a mouse action (click or move), Sigma will automatically switch back to English.

This mode is suitable for software that requires fast interaction and mainly uses English — for example, shooting games, programming, or technical applications. When you need to type Vietnamese (e.g., sending a message), you can quickly switch to Vietnamese. Once a mouse action occurs, Sigma will return to English to avoid confusion.

Gõ tắt tức thì

You can choose one of the predefined shorthand lists. The default is Không sử dụng, in which case shorthand is disabled.

With gõ tắt tức thì, words are automatically replaced as soon as the typed sequence matches — without pressing space or punctuation.

Suitable for quickly inserting special symbols or frequently used abbreviations, such as φ, π, σ, √.

Gõ tắt khi cách

You can also select one of the available shorthand lists. The default is Không sử dụng.

With gõ tắt khi cách, replacement only occurs when you press space or punctuation, and the typed sequence matches the list.

Suitable for longer phrases or templates — giving you precise control over when replacements happen.

Formatting rules for shorthand results (applies to both modes):

  • If the shorthand entry is all lowercase → result keeps the same formatting as in the list.
  • If the first character is uppercase → result is in Title Case.
  • If the second character is uppercase → result is in UPPERCASE.

Shorthand results will be automatically converted according to the current character set, such as Unicode, VNI Windows, or TCVN3, ensuring compatibility with the software or document you are working on.

You can create multiple shorthand lists for different software or purposes. To edit, click the Sigma icon in the system tray (bottom-right corner), then select Trình quản lý gõ tắt.

Sử dụng clipboard khi gửi phím

When this option is enabled, Sigma sends text by copying the content to the clipboard and pasting it, instead of simulating keystrokes.

This option is especially useful for applications that do not support Unicode or fail to handle Vietnamese input via normal keystrokes.

For applications detected as not supporting direct Unicode input (e.g., AutoCAD), Sigma will automatically enable this option by default the first time you use it or after resetting settings. Users can still disable it if desired. If disabled, Sigma will display this option in red along with a warning that it may cause typing errors in some cases with the current application.

Note: This option may temporarily overwrite the current clipboard content. However, Sigma can automatically restore the original clipboard content after the operation is complete, minimizing data loss.

Tương thích với ứng dụng có gợi ý từ

When enabled, Sigma adjusts keystroke handling to ensure stable operation in applications with word suggestions.

For applications prone to issues when using word suggestions (e.g., web browsers, Excel, messaging apps, code editors), Sigma will automatically enable this option by default the first time you use it or after resetting settings. Users can still disable it if desired. If disabled, Sigma will display this option in red with a warning that it may cause typing errors in some cases with the current application.

Note: Enabling this helps avoid conflicts and improve accuracy when typing. However, in some cases it may slightly slow down input response. Some users may prefer to disable word suggestions in their applications — in that case, enabling this option in Sigma is not necessary and you can turn it off, ignoring the warning.

Tự động viết hoa thông minh

This feature automatically capitalizes the first letter of a sentence or after punctuation without needing to hold Shift.

  • When starting a new line or after punctuation such as ., !, ?, the first character of the next sentence will be capitalized automatically.
  • If you capitalize the second character of a word (e.g., vIệt), Sigma interprets it as a request to capitalize the entire word, and it will automatically change to VIỆT.

This saves time when typing quickly while maintaining correct spelling rules. It is especially useful for writing documents or chatting.

Cho phép dùng phụ âm đầu "f", "j", "w", "z"

When this option is enabled, you can type Vietnamese words with initial consonants like f, j, w, z without being treated as spelling errors.

If you also enable "Khôi phục từ gốc khi gõ sai chính tả", the system will not revert to the original word in cases where you intentionally type words such as:

  • zui wá
  • zậy sao
  • jì thế

To allow more non-standard words, you should disable "Khôi phục từ gốc khi gõ sai chính tả".

Dùng phím ← → để điều hướng từng từ

Allows using arrow keys to navigate through segments of the current word/phrase, making quick edits easier. However, this option is not recommended in code editors (e.g., when programming), since word-based navigation with arrow keys may interfere when editing nested strings or code syntax.

Kiểu gõ

You can choose between common input methods such as Telex, VNI, or Tích hợp.

Khởi động cùng Windows

Automatically launch Sigma whenever Windows starts, so you don’t need to open it manually each time.

Khởi động Sigma với quyền quản trị (Admin)

By default, Sigma runs with normal user privileges. In this mode, it cannot interact with applications running under administrator rights.

When this option is enabled, Sigma will run with full administrator privileges, allowing it to work normally with all applications, including those requiring admin rights.

Note: This change only takes effect after you restart Sigma.

This feature is not available in older versions and cannot be added through updates. If you don’t see this option in the configuration, please uninstall and reinstall Sigma to use it.

Khôi phục từ gốc khi gõ sai chính tả

This option is useful if you often mix Vietnamese and English. When enabled, Sigma automatically switches to English if it detects that the typed word is not valid Vietnamese.

However, if you need to type special cases such as abbreviations or place names with glued diacritics (e.g., Đắk Lắk), it’s better to disable this option.

Example:

  • N Đ C P → NĐCP
  • Đắ k Lắ k → Đắk Lắk
  • Đắ k G long → Đắk Glong

In such cases, Sigma interprets your input as valid glued Vietnamese (e.g., Đắ + k → Đắk). You need to add diacritics at the correct time. Once you move to the next word, you cannot add accents to the previous word.

Tip: Assign a shortcut to this option so you can toggle it quickly when needed.

Tự động chuyển từ tiếng Anh đã ghi nhớ

When this option is enabled, Sigma remembers English words that you manually switched using Left Ctrl or by pressing a diacritic key twice. Later, when you type those words again, Sigma will automatically switch to English mode.

This feature is especially useful if you use Telex or Tích hợp, where English words can be mistaken for accented Vietnamese — for example:

    t e s t → tét

When such misinterpretations occur, press Left Ctrl or a diacritic key to switch to English. If you press space or punctuation to finish the word, Sigma will remember it as English and handle it correctly in the future.

If later you want to type it as Vietnamese again, press Left Ctrl once more or Backspace. Sigma will revert to Vietnamese and remove the word from the memory list.

  • t e s s t → test
  • Or t e s t → tét press Left Ctrl → test
  • Word test saved; next time typing t e s t → test
  • t e s t → test press Left Ctrl or Backspace → tét (word test removed from memory)

Sigma stores memory lists per application to ensure words are typed correctly in context.

Example:

  • In AutoCAD: typing e x → ex (auto-switched to English)
  • In another app: typing e x → ẽ (kept as Vietnamese)

Xóa toàn bộ dấu khi nhấn phím bỏ dấu

When this option is enabled, each time you press the remove-diacritic key, Sigma will remove all diacritics from the current word, including:

  • Diacritics on characters: â, ă, ê, ô, ơ, ư, đ
  • Tone marks: sắc, huyền, hỏi, ngã, nặng

Example when enabled (using Telex):

  • "Đất", type d → "Datd"
  • "Đất", type a → "Data"
  • "Đất", type s → "Dats"
  • "Đất", type z → "Dat"

Example when disabled:

  • "Đất", type d → "Dấtd"
  • "Đất", type a → "Đáta"
  • "Đất", type s → "Đâts"
  • "Đất", type z → "Đât"

When disabled, this behaves like other common Vietnamese input methods: only removing the last tone mark or accent, while keeping the others intact.

This is an advanced option unique to Sigma, suitable for users who want to quickly convert a word to non-accented form with just one key press.

Cho phép bỏ dấu xoay vòng

When enabled, you can repeatedly press tone keys such as s, f, r, x, j to cycle states:

  • No diacritic → With diacritic → No diacritic

Example: type a s → á, type s again → a.

Useful for quickly correcting diacritics without deleting and retyping.

For new Sigma users unfamiliar with switching languages via Ctrl, you may disable this option to keep the traditional method: pressing the tone key twice switches to English.

Example: type e x → ẽ, type x again → ex (removes diacritic and keeps the base letter).

Cho phép chèn ký tự bị thiếu

If you miss a character → just add it → Sigma will automatically insert it at the correct position and rearrange the syllable. This also works when characters are typed in the wrong order due to fast typing — Sigma will adjust if the word is still recognized as valid.

Example: Type h o n g, then add a → hoang.

Very useful for fast typists, reducing spelling errors while maintaining smooth typing speed. However, when enabled, Sigma becomes highly sensitive to Vietnamese input, which may be inconvenient in apps with lots of English typing.

Therefore, this option is designed to be configured per-application. You are encouraged to enable it in Vietnamese-focused apps (word processors, social media…) and disable it in English-heavy apps (IDE, technical browsers).

Note when enabling enhanced features

The following features are developed by Sigma to make Vietnamese typing more flexible:

  • Xóa toàn bộ dấu khi nhấn phím bỏ dấu
  • Cho phép bỏ dấu xoay vòng
  • Cho phép chèn ký tự bị thiếu

However, when these are enabled, if you are in Vietnamese mode and type English words, they may be unintentionally transformed. In this case, you cannot switch to English by the traditional method of pressing the tone key twice. Instead, you must use Left Ctrl to switch the current word to English.

This is a unique Sigma feature — not available in other input methods — so new users may need time to adapt. These advanced features should only be enabled once you are comfortable with using Left Ctrl for language switching during typing.

When enabled, Sigma will automatically display the related option text in red along with a warning, to alert users that it may cause difficulties for beginners or those who prefer traditional typing. This helps users be aware and make an informed choice.

7. Integrated AI Tasks

AI Tasks

AI tasks help you quickly process text while typing, such as spell checking, translation, or character set conversion — all without leaving the keyboard.

Default Tasks:

  1. Kiểm tra chính tả tiếng Việt
  2. Kiểm tra chính tả tiếng Anh
  3. Dịch sang tiếng Việt
  4. Dịch sang tiếng Anh
  5. Chuyển mã Unicode

Practical Applications

  • Write emails, reports, and content quickly and professionally.
  • Rewrite paragraphs, summarize, or change writing style.
  • Quickly translate documents or convert legacy text encodings.

Customizing AI Tasks

  • Right-click the Sigma icon in the system tray.
  • Select Trình chỉnh sửa tác vụ AI.
  • You can add tasks such as: Tóm tắt nội dung dài, Viết lại đoạn văn, Chuyển thành văn nói hài hước, and more.

Note: Default tasks cannot be edited or removed.

Quick Access with Shortcut Keys

AI task names are sorted alphabetically. Shortcuts Shift → F1 through Shift → F12 will trigger tasks 1 to 12 in the list.

To keep the order consistent and easier to remember, it’s recommended to add a numeric prefix before task names (e.g., 01. Kiểm tra chính tả tiếng Việt, 02. Kiểm tra chính tả tiếng Anh, …). This ensures:

  • A fixed order in the list.
  • Easy mapping to the corresponding Shift → F# shortcuts.

How to Use Shortcuts:

  • Users can select text with the mouse, then press the corresponding shortcut key to trigger the desired AI task.
  • If continuously typing, Sigma will automatically store the current text in temporary memory. Users can press the shortcut key without selecting text, and the AI task will be applied to the stored text.
  • Note: If you click or type, the temporary memory is cleared. In that case, you must manually select text before using an AI task.

Receiving Results from AI Tasks

Users can call AI tasks with either Left Shift or Right Shift. Depending on the context, the results will be handled differently:

  • Left Shift: The result will replace the selected text directly. This works only in editable text fields (editors, input boxes, writing areas, etc.).
  • Right Shift: For non-editable contexts (file names, menus, titles…), the result will appear in a separate interface panel for you to review or copy manually.

10. Other Issues

Notes when typing in Notepad

When using Notepad on Windows 11, if Spell check is enabled, you may encounter Vietnamese typing errors: missing characters, misplaced characters, or incorrect input.

To type Vietnamese correctly in Notepad, Spell check must be turned off. Currently, there is no other solution.

How to disable Spell check in Notepad (Windows 11):

  • Open Notepad.
  • Click the gear icon (Cài đặt / Settings) in the top-right corner.
  • Find Spell check (or Kiểm tra chính tả).
  • Turn off the Spell check toggle.

Using Gemini AI in Sigma

AI tasks in Sigma are performed by connecting to the internet and sending API requests to the Gemini AI system. Sigma comes with pre-integrated Gemini API keys, updated regularly to serve users.

However, if too many users are active at the same time, some keys may be overloaded and unable to process AI tasks.

For stable and independent usage, you can create your own Gemini API key and add it to Sigma by:

  • Click the Sigma icon in the system tray (bottom-right corner).
  • Select Trình quản lý tác vụ AI.
  • Click the Khóa API Gemini button, then enter your key.

You can create a Gemini API key here:

  • Create Gemini API Key

It is recommended to add multiple API keys. Sigma will randomly pick one key from the list for each AI task, reducing the risk of hitting usage limits.

Create auto-control utilities for Sigma per application

Sigma allows external software to automatically control English/Vietnamese mode and character set via Registry keys. You can use this feature to build small tools or integrate with your own software, eliminating the need for manual switching.

All control is done through the key: HKEY_CURRENT_USER\Software\Sigma\AutoSigma

AutoCAD integration with Auto Sigma.lsp

Sigma supports automatic switching of English/Vietnamese mode and character set when creating or editing TEXT / MTEXT objects in AutoCAD. This is enabled via an AutoLISP utility named Auto Sigma.lsp.

Download Auto Sigma.lsp

Main features:

  • Whenever you create or edit text objects, Sigma will:
    • Analyze the object’s font to automatically choose the correct character set (Unicode, TCVN3, VNI...)
    • Switch to Vietnamese mode so you can type with diacritics correctly.

Usage tips:

  • Add Auto Sigma.lsp to the Startup Suite in AutoCAD so it runs automatically.
  • Type the command ASIG in AutoCAD to change Sigma’s auto settings anytime.

With this feature, you no longer need to manually switch character sets or input modes when working with multiple fonts in AutoCAD.

When the above Registry values are updated, Sigma automatically detects and adjusts the typing mode accordingly. This is especially useful when you want to quickly change typing settings for a specific application.
